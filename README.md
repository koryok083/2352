## Installation

1. Download APK from [Releases](https://github.com/koryok083/android-modem-app/releases)
2. Enable "Unknown Sources" in Settings
3. Install APK
4. Grant required permissions
5. Run the app

## Requirements

- Android 7.0+ (API 24+)
- Root access (recommended)
- 50MB free storage
